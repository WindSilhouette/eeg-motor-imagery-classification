# EEG-Based Interpretable Motor Imagery Classification

This project explores interpretable deep learning models for decoding motor imagery and motor execution from EEG signals using the PhysioNet EEG Motor Movement/Imagery dataset.

## Project Overview
- Dataset: PhysioNet EEG Motor Movement/Imagery
- Models: CNN-LSTM-Attention, CNN-Transformer, CNN-LSTM-Transformer
- Interpretability: Grad-CAM, Saliency Maps
- Metrics: AUC, F1, Balanced Accuracy
